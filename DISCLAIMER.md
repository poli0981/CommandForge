## ⚠️ Use at your own risk

CommandForge **executes system-level commands** on Windows. Using it can:

- Change system configuration (registry, services, Windows features…).
- **Cause data loss or an unstable system** if used incorrectly or in the wrong conditions.
- Require a restart to finish some operations.

**You are solely responsible** for every command you choose to run and for any consequences.

## Safety recommendations

- **Back up** important data before running risky commands.
- **Create a System Restore Point** before `Caution`/`Dangerous` operations (the app offers this).
- Read each command's **description and preview** before running it.
- Don't run commands you don't understand.
